# Godot RPG Project:

## Controls:
| Action | Keyboard | Gamepad (Xbox Labels) |
|:------:|:--------:|:---------------------:|
|  Walk  |   WASD   |     L Stick, D-Pad    |
| Camera |  Arrows  |        R Stick        |
|  Jump  |   Space  |           A           |
|   Run  |   Shift  |           B           |
